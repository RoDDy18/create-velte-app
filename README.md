## Installation & Usage

* Install CLI globally

```bin
npm i -g create-velte-app
```

* Create test app with default arguments

```bin
mkdir test-app
cd test-app
create-velte-app --yes --git --install
```

* or Run CLI without installation

```bin
npx create-velte-app --yes --git --install
```
